# Basketball Analysis System

A computer vision project for detecting basketball players and sports balls in images and videos using YOLO object detection.

## Features

- Detect players in images and videos
- Detect sports ball/basketball using YOLO
- Supports image and video input
- Saves annotated detection output
- Can be extended with a custom trained basketball model

## Tech Stack

- Python
- Ultralytics YOLO
- OpenCV
- Roboflow
- Computer Vision
- Deep Learning
